Prevent sql injection with lists in python
3 Martijn Pieters (1,136,906 rep) 2015-12-07
Answer Explanation:

You need to pass in a tuple: Note the comma! You can also pass in a list: Since you have a list of values to insert, you can use cursor.executemany() to have the database apply all values in one go, but you need to make each element in your list a tuple or list: Note that you can use both the connection and cursor as a context managers. When the with block for the connection ends, the transaction is automatically committed for you, unless an exception has occurred at which point the transaction ...

text
database.execute(command, (data,))
text
database.execute(command, [data])
sql
with connection:
    with connection.cursor() as cursor:
        command = "INSERT INTO my_table(my_value) VALUES %s"
        cursor.executemany(command, ((data,) for data in mylist))
